# Github & Publishing

* Github의 3 + 1 가지의 기능
  * 코드 저장 기능
  * Undo 기능
  * 협업 기능
  * Web Hosting 기능
    - Easy & Free



* Web Hosting(Publishing) 과정
  1. 새 저장소(Repository) 생성
  2. 이름 반드시 '<id>.github.io'로 설정 후 저장소 생성
  3. 원하는 파일(index.html) 업로드
  4. 코멘트 달아서 commit(파일 밀어넣기)
  5. github page 배포 여부 확인(url : '<id>.github.io')



* github는 index.html 파일을 제일 먼저 인식
