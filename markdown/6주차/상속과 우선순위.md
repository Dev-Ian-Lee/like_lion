<h1>상속과 우선순위</h1>

* 상속(inherit) : 부모의 속성을 자식도 갖는 것

  * 모든 CSS 프로퍼티가 상속되는 것은 아님

  * 상속되지 않는 프로퍼티는 속성값을 inherit으로 설정하면 상속 가능

    * margin: inherit;

      

* 우선순위

  * 규칙

    * 중요도

    * 명시도

      * !important 써주면 1순위

      * id 선택자 사용 지양

        

    * 선언 순서 : 나중에 선언된 스타일이 우선 적용(Cascading)
